# E-Book - Responsive Book Website

A modern, responsive, and fully interactive book store landing page built with HTML, CSS, and Vanilla JavaScript. This project features a clean UI, engaging animations, and a seamless user experience across all devices.

## âœ¨ Features

- **Responsive Design**: Optimized for desktops, tablets, and mobile phones.
- **Dark/Light Mode**: Toggle between light and dark themes with persistent storage using `localStorage`.
- **Interactive Swipers**: Dynamic book carousels for the Hero, Featured, New Books, and Testimonial sections (powered by [Swiper.js](https://swiperjs.com/)).
- **Search & Login Modals**: Sleek, overlay-based search and login interfaces.
- **Scroll Animations**: Smooth entrance animations for sections as you scroll (powered by [ScrollReveal.js](https://scrollrevealjs.org/)).
- **Modern UI Components**:
  - Floating navigation menu with shadow on scroll.
  - Newsletter subscription section.
  - Detailed book cards with price tags and action buttons.
  - Testimonial section with client reviews and star ratings.
  - Scroll-to-top button.
